# Plataforma de treinos 🏋️

Este projeto é uma plataforma web de fitness que gera treinos e planos alimentares personalizados para usuários, tanto para academia quanto para treinos em casa.
A aplicação busca oferecer uma experiência completa de acompanhamento físico, unindo tecnologia, educação e orientação profissional

## Visualizar no GitHub Pages 🌐
A plataforma está hospedada no GitHub Pages! Você pode acessá-lo diretamente através deste link: https://gbriellnunes.github.io/Plataforma-de-Treinos/

## Objetivo 🎯

Este projeto para portfólio foi criado com foco em prática de desenvolvimento front-end, organização visual, responsividade e construção de interfaces mais interativas.

## Funcionalidades 🚀

- página inicial com apresentação da plataforma
- navegação entre seções
- área de login
- área de cadastro
- exibição de planos
- interações com JavaScript
